Objectif global du système

Trier automatiquement 4 types de déchets (cubes colorés : vert, jaune, rouge, bleu) détectés sur un convoyeur, avec suivi en temps réel via une interface web.

Description du fonctionnement du système

Fonctions principales

Fonction Description
Détecter la présence d'un déchet Utilise un capteur pour activer le convoyeur
Détecter la couleur du déchet Capteur de couleur identifie la couleur du cube
Classer le déchet détecté Indique dans quelle benne manuelle placer le déchet
Compter les déchets triés par couleur Incrémentation du compteur selon la couleur
Afficher les statistiques sur une interface web Envoi des données de l'ATMéga vers interface pour visualisation live
Alimenter le système de façon autonome Bloc batterie Lithium alimente l'ATMéga, capteurs et moteur

Contraintes techniques à respecter

  • Dimensions du convoyeur : 650 mm de long, 100 mm de haut
  • Déchets : Cubes de 30 mm
  • Carte : Arduino Nano ou ATmega328P
  • Alimentation : batterie lithium uniquement
  • Pas de breadboard dans le rendu final
  • Interface web avec affichage en temps réel

Architecture système

Voici les 5 grands blocs du système :

  1. Bloc d'alimentation
  2. Bloc détection (Capteur de présence, Capteur de couleur)
  3. Bloc convoyeur (Moteur avec driver, Bande de transport)
  4. Bloc traitement (Microcontrôleur, Lecture des capteurs, Traitement couleur + comptage)
  5. Bloc interface web (Page web responsive avec 4 compteurs)

DÉFINITION DES BESOINS TECHNIQUES

1. Électronique

Objectifs techniques :

  • Détecter la présence d'un déchet
  • Identifier sa couleur
  • Commander le moteur du convoyeur intelligemment
  • Indiquer la couleur détectée (LED)
  • Envoyer les données de tri vers l'interface web
  • Alimenter tout le système proprement

CHOIX ET JUSTIFICATION DES COMPOSANTS

Composant choisi Justification technique
ATmega328P Petit, léger, facilement intégrable sur PCB
Laser KY-008 + Photorésistance Système simple et fiable, facile à intégrer mécaniquement
TCS3200 Capteur de couleur RGB avec sortie fréquence, fiable pour distinguer 4 couleurs
Moteur DC 6–12V L298N permet de piloter le moteur DC avec protection thermique
Batterie Lithium + Régulateur 5V Batterie recommandée pour autonomie

2. Informatique

Objectifs techniques :

  • Créer une interface web intuitive, responsive, en temps réel
  • Afficher le nombre de déchets triés (par couleur)
  • Recevoir les données de l'Arduino via communication série
  • Intégrer les logos TEKBOT et TRC 2025

Technologies :

Fonction Outil/Techno Détail
Interface web HTML/CSS/JS Simple, responsive
Communication JS + WebSerial API ou backend Node.js/Python Flask Pour recevoir les données depuis atmega
Mise à jour live JavaScript DOM ou Socket.IO Affichage des compteurs couleur en temps réel

⚙️ 3. Mécanique (SolidWorks / Convoyeur / Support capteurs)

Objectifs techniques :

  • Concevoir le convoyeur de 650 mm x 100 mm
  • Intégrer support moteur, rouleaux, tapis, capteurs
  • Prévoir les emplacements pour composants électroniques
  • Réaliser l'assemblage final solide et fonctionnel
